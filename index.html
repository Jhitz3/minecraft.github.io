Minecraft

Minecraft is a sandbox game that released in 2011. With over 300 million sales, Minecraft is the worlds best selling game of all time. Although Minecraft can be enjoyed by all ages, it can be increasingly complex, which is why I created this website. This website will help teach you some of the basics of Minecraft so that you can get a head start when playing the game. This website will explore a few different topics which include: Crafting, Mobs, Biomes, and Dimensions.
